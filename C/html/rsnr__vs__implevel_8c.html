Detailed Description

C routine that generates the data for building the Figure 2(c).

Figure 2(c) shows the curves of the reconstruction signal-to-noise ratio (R-SNR) in dB of the recovered coefficient vector versus the impulsiveness level of the noisy projections. For this experiment, the additive perturbations are characterized using an -contaminated noise model (also known as p-Gaussian noise) with . The variance of the Gaussian noise is computed in order to obtain a previously defined SNR value, i.e. . Furthermore, this experiment obtains the SNR values generated by both the proposed sparse signal recovery algorithm (WMR-AR, weighted median regressions with adaptive regularizer) and the method based on weighted median regressions with hard thresholding (WMR-HT). Each point in the curves are obtained by averaging a user-defined number of trials, which is introduced in running time. The results of the simulation are saved in the results directory and the curves can be visualized by running the Figure2c.m routine. This plotting routine can be executed under either Octave or Matlab.
